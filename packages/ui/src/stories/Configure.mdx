# GlueOS UI Kit

Welcome to the GlueOS UI Kit — a comprehensive, production-ready component library built for modern data-driven applications. This design system provides everything you need to build consistent, accessible, and beautiful user interfaces across the GlueOS ecosystem.

## What is the GlueOS UI Kit?

The GlueOS UI Kit is a curated collection of React components, patterns, and design tokens that accelerate development while maintaining design consistency. Built on industry-standard technologies (React, TypeScript, Tailwind CSS, and Radix UI), it provides:

- **60+ Production Components** — From basic buttons to complex data visualizations
- **Themeable Design System** — Fully customizable colors, typography, spacing, and corner radius
- **Accessibility Built-In** — WCAG AA compliant components with keyboard navigation and screen reader support
- **Pattern Library** — Pre-built authentication flows, dashboard layouts, and data visualization patterns
- **Client Software Generator** — Automated tooling to create branded instances for clients

## How to Use This Storybook

This documentation serves multiple purposes:

### 1. Component Reference
Browse through **Foundation**, **Components**, and **Patterns** to explore the full library. Each component includes:
- Live interactive examples

### 2. Theme Customization Preview

For detailed brand customization, visit **[0-Defaults → Configurator](/story/0-defaults-configurator--default)**:
- Visual neutral palette selector (5 gray scales)
- Individual brand color pickers (Primary, Secondary, Accent)
- Visual chart palette selector (17 chromatic options)
- Separate heading and body font controls
- Corner radius slider (sharp to rounded)
- Logo upload for all 4 variants
- Live preview panel showing real components

### 3. Client Software Generator

The Configurator mirrors the interface used in the Client Software Generator web app. When you generate a client project:

1. Configure brand settings (colors, fonts, logos, radius)
2. Generator creates a custom monorepo with all settings applied
3. CSS variables are automatically injected into the theme system
4. All components inherit the client's brand instantly

## Design System Structure

The GlueOS UI Kit is organized into three layers:

### 1. Foundation
Core design tokens and primitives that form the base of the system:
- **Colors** — Neutral palettes, brand colors, semantic colors, and chart palettes
- **Typography** — Font families, type scales, and text utilities
- **Spacing** — Consistent spacing scale from 0.125rem to 20rem
- **Shadows** — Elevation system for depth and hierarchy
- **Logos** — Brand assets (light, dark, icon, favicon variants)

### 2. Components
60+ reusable UI building blocks:
- **Form Controls** — Inputs, selects, checkboxes, radio groups, sliders
- **Navigation** — Menus, breadcrumbs, tabs, pagination
- **Feedback** — Alerts, toasts, progress indicators, skeletons
- **Overlays** — Dialogs, drawers, popovers, tooltips
- **Data Display** — Tables, cards, badges, avatars, code blocks
- **Utility** — Buttons, separators, scrollable areas, containers

### 3. Patterns
Pre-composed patterns for common use cases:
- **Authentication** — Login forms, signup flows, password reset
- **Data Visualization** — Interactive charts (line, bar, area, pie, radar, scatter, gauge)
- **Dashboard Layouts** — Application shells, responsive grids

## Customizable Theme System

| Setting | Options | Applied To |
|---------|---------|------------|
| **Neutral Palette** | 5 gray scales (slate, gray, zinc, neutral, stone) | Backgrounds, borders, muted text |
| **Brand Primary** | Any hex color | Buttons, links, primary actions |
| **Brand Secondary** | Any hex color | Secondary buttons, accents |
| **Brand Accent** | Any hex color | Highlights, call-outs, badges |
| **Chart Palette** | 17 chromatic palettes | Data visualizations, analytics |
| **Heading Font** | 8 font families | All heading levels (H1-H6) |
| **Body Font** | 8 font families | Body text, UI copy, paragraphs |
| **Corner Radius** | 0rem to 1rem | Buttons, cards, inputs, borders |
| **Logos** | 4 SVG/PNG variants | Navigation, headers, favicons |

## Getting Started

### Explore Components
Navigate through the sidebar to browse the full component library. Each story includes interactive examples and comprehensive documentation.

### Try Theme Customization
1. Visit the **[Configurator](/story/0-defaults-configurator--default)** for full control
2. Upload custom logos and see them reflected in the live preview
3. Experiment with different color combinations and fonts

### Use in Development
The GlueOS UI Kit is available as an npm package (`@repo/ui`) in the monorepo workspace. Import components directly into your application:

```tsx
import { Button, Card, ChartAreaInteractive } from '@repo/ui'
```
